# Design System

## Design Philosophy

The interface should feel:
- calm
- soft
- ambient
- breathable
- elegant
- emotionally gentle

Avoid:
- clutter
- dashboard feeling
- aggressive colors
- enterprise UI
- dense information

---

# Visual Language

## Style

- iOS native glassmorphism
- soft blur
- layered translucency
- large spacing
- smooth rounded corners
- minimal visual noise

---

# Colors

Primary:
- soft white
- translucent gray
- muted blue
- subtle cyan glow

Avoid:
- saturated colors
- harsh contrast
- neon UI

---

# Typography

- clean
- spacious
- Apple-like hierarchy

Avoid:
- tiny text
- dense layout

---

# Animation

Animations should feel:
- breathing-like
- soft
- organic
- smooth
- slow enough to feel calm

Avoid:
- bouncy excessive motion
- flashy transitions

---

# UI Principles

Before adding elements:
ask whether the UI can become simpler.

Prefer:
- emotional atmosphere
- focus
- silence
- whitespace

over:
- information density
- productivity feeling
- dashboard complexity
